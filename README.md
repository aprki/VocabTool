# VocabTool

VocabTool 是一款用於英語閱讀課與自學的單字練習工具，使用者可上傳自訂單字表，快速建立測驗網站，並透過內建的複習機制系統性地鞏固詞彙。工具支援離線練習、匯出詞庫與多樣化學習模式，設計上不依賴外部 API，以降低資料外流風險並維持介面簡潔。

## 主要功能
- 上傳單字表：支援 CSV / TSV 等常見格式，快速匯入詞彙清單，建議欄位包含英文單字、詞性、中文註解與例句。
- 自動建立測驗網站：依據上傳的單字表生成互動式練習頁面，提供即時檢測與成績回饋。
- 複習機制（Review）：內建複習流程，依學習情況安排回顧以強化長期記憶。
- 離線練習與抽認卡：支援無網路環境下的練習模式，包含抽認卡學習與多種題型的自我測驗。
- 匯出功能：可匯出整體詞庫或個人學習記錄，用於備份或移轉至其他平台。

## 適用對象
- 英語教師：便於為課堂或班級建立專屬的單字練習站，改善作業布置與學習追蹤流程。
- 學生與自學者：以自有單字表打造個人化練習內容，搭配複習機制提升學習效率。
- 小型教學單位或讀書會：部署門檻低、操作直覺，適合作為團體或小班教學的輔助工具。

## 快速上手
1. 準備單字表（CSV/TSV），建議包含英文單字、詞性、中文註解與例句欄位。  
2. 在網站上上傳單字表並建立詞庫。  
3. 使用「自我測驗」或「複習」模式進行學習；可視需求匯出詞庫或學習紀錄。
