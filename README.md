# 單字自我測驗工具 · Vocab Master

一組不需安裝、不需網路（除了語音朗讀功能）、雙擊即可開啟的單機版 HTML 單字複習工具。適合個人背誦，也可以直接分享給學生或團隊使用。

## 檔案總覽

| 檔案 | 詞庫內容 | 功能 |
|---|---|---|
| `vocab_flashcard_quiz.html` | GRE 高階單字（43 個 List，共 2,538 字） | Flashcard 卡片背誦 + 選擇題測驗 |
| `ielts_pte_vocab_tool.html` | PTE 學術搭配詞（2,468 組）＋ IELTS 1200 聽力單字（925 字，35 個主題） | Flashcard／選擇題（PTE）＋ 拼寫聽力測驗（IELTS） |

兩個檔案完全獨立，**下載後直接用瀏覽器打開就能用**（Chrome、Edge、Safari 皆可），不需要伺服器、不需要安裝任何東西。

---

## 共通功能

- **範圍選擇**：可勾選任意 List／分類，也可以「全選」「清空」或「隨機挑幾組」
- **Flashcard 模式**：點卡片翻面看解釋，可標記「記住了 ✓ / 不熟 ✗」、上一個／下一個、切換隨機順序
  - 鍵盤快捷鍵：`空白鍵` 翻面、`←/→` 切換上一個／下一個
- **選擇題測驗模式**：四選一，答題後立即顯示對錯，並附上英文定義加深印象；測驗結束會列出所有答錯的單字方便複習
- 所有進度、分數僅存在當次瀏覽器分頁中，**重新整理頁面會重置**，不會自動存檔

## `vocab_flashcard_quiz.html`（GRE 版）

- 資料來源：GRE 高頻單字表，共 43 個 List
- 每個單字包含：詞性、繁體中文釋義、英文定義（取自 WordNet 詞典資料庫）
- 適合每天挑幾個 List 做隨機複習或選擇題自我測驗

## `ielts_pte_vocab_tool.html`（IELTS / PTE 版）

打開後會先看到「詞庫」選擇，切換後下方選項會跟著變化：

### 📚 PTE 學術搭配詞
- 資料來源：Pearson *The Academic Collocation List*，共 2,468 組 collocations
- 依字母 A–Y 分組
- 功能與 GRE 版相同：Flashcard（含中譯、英文定義、例句）＋ 選擇題測驗（可選「看搭配詞選中文」或「看中文選搭配詞」兩種方向）

### 🎧 IELTS 1200 聽力拼字
- 資料來源：IELTS Listening 常考單字表，依主題分成 35 組（Days、Hobbies、Health、Transportations…）
- **拼寫聽力測驗**（新功能）：
  1. 系統用瀏覽器內建語音朗讀單字
  2. 可點喇叭 🔊 重播，或切換「🐢 慢速發音」
  3. 在欄位輸入你聽到的拼寫，按 `Enter` 或「送出答案」
  4. 立即顯示對錯；答錯會顯示正確拼法
  5. 測驗結束顯示正確率，並列出所有拼錯的單字（含你當時打的答案）方便複習

> ⚠️ 拼寫測驗需要瀏覽器支援 Web Speech API（`speechSynthesis`）。Chrome、Edge、Safari 最新版都支援；語音音色會依你電腦/手機系統內建的語音引擎而不同。若瀏覽器完全不支援語音，喇叭按鈕會沒有聲音，但仍可手動練習拼寫。

## 怎麼用 / 怎麼分享

- **個人使用**：直接雙擊檔案，用瀏覽器打開即可
- **分享給學生或團隊**：
  1. 把 `.html` 檔案傳給對方（Email、雲端硬碟、LINE 皆可），對方下載後雙擊開啟即可，不需要任何額外安裝
  2. 或上傳到 GitHub Pages／Google Drive 等靜態託管服務，取得連結後直接分享網址
